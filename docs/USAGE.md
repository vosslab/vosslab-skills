# Usage

## Primary path

1. Open a skill directory under `skills/`.
2. Read that skill's `SKILL.md`.
3. Follow the workflow and constraints in that file for the current task.

## Example

```bash
cd /Users/vosslab/nsh/vosslab-skills
ls skills
cat skills/readme-fix/SKILL.md
```

Use only the skills relevant to the active request.
